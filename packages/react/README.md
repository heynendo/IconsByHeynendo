# Icons by Heynendo

SVG icon library for React.

## Installation

```bash
npm install icons-by-heynendo
```

## Usage

```jsx
import { Menu, ArrowDown } from 'icons-by-heynendo';

function App() {
  return (
    <div>
      <Menu size={24} color="#000" />
      <ArrowDown size={20} rotation={90} />
    </div>
  );
}
```

## Props

- `size` - number | string (default: 24)
- `color` - string (default: 'currentColor')
- `rotation` - number in degrees (default: 0)
- `className` - string
- `style` - object
